Задача # 4


Написать на любом языке: Python,js,bash,powershell скрипт, который будет выдавать метрику загрузки процессора Linux
Решение. Cкрипт, при вызове, в консоль  должен выводить строку в формате flux line protocol измеряемой метрики
Оценка: 5
Дополнительные сведения:
Выводить просто busy в %

Ответ:
Смотрите файлы mem_usage_fluxline.js,  mem_usage_fluxline.py, mem_usage_fluxline.sh и, прости Господи,  mem_usage_fluxline.ps1 в этой директории. 
Скрипт do_all.sh поочередно запускает каждый из скриптов. Для работы на целевой linux-машине необходимы, соответственно, python, bash, node.js и интерпритатор powershell для linux.
