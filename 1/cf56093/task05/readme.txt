Задача # 5


Написать на любом языке: Python,js,bash,powershell скрипт, который будет выдавать метрику загрузки процессора Windows
Решение. Cкрипт, при вызове, в консоль  должен выводить строку в формате flux line protocol измеряемой метрики
Оценка: 5

Ответ:
Смотрите соответствующие файлы скриптов в этой директории. Переработке подвергся только скрипт на powershell, который вместо парсинга вывода утилиты top использует специальный объект Win32_Processor, который, разумеется, не существует в Linux. Скрипт do_all.bat поочередно запускает каждый из скриптов. Для работы на целевой windows-машине необходимы, соответственно, python с модулем psutil, bash из WSL2, node.js и powershell с разрешенной системной политикой выполнения скриптов. По причинам, указанным ещё в первом задании, у меня не было возможности протестировать Bash-скрипт в windows-среде, но он должен работать корректно, если WSL действительно предоставляет все возможности Linux.
